# GEMINI.md

## Project Overview

This is a Unity project for a Tic Tac Toe game. The project is in its initial phase and contains the basic structure of a Unity project.

*   **Engine**: Unity 6000.3.6f1
*   **IDE**: Visual Studio
*   **Language**: C#

## Building and Running

1.  **Open the project in Unity**:
    *   Open Unity Hub.
    *   Click on "Open" and select the root directory of this project.
    *   Unity will load the project.

2.  **Run the game**:
    *   Open the `SampleScene` from the `Assets/Scenes` directory.
    *   Click the "Play" button in the Unity Editor.

## Development Conventions

*   **Scripts**: All C# scripts should be placed in the `Assets/Scripts` directory.
*   **Scenes**: All scenes should be placed in the `Assets/Scenes` directory.
*   **Assets**: All other assets (images, models, etc.) should be organized in subdirectories within the `Assets` directory.
